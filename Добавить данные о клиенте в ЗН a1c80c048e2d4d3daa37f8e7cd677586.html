Добавить данные о клиенте в ЗН

Контекст задачиВ документе “заказ-наряд” должны быть зафиксированы данные о клиенте. Если клиент уже обращался на СТО, то пользователь может выбрать данные клиента из справочника клиентов.
Заинтересованные стороныМастер СТО
Ключевые источники информацииТребования к системе
Атрибуты качества и ограничения (НФТ)

Критерии приемки результата.

  1. Создан справочник клиентов.
  1. При введение фамилии и имени клиента в строку поиска система выводит на экран список всех совпадений.
  1. Создана карточка клиента.
  1. При выборе конкретного клиента система выводит на экран все данные из карточки клиента.
  1. Пользователь может вносить изменения в карточку клиента

Описание решения

Приложение 1 UC “Добавить данные о клиенте в заказ-наряд”

Основной сценарий:

  1. Мастер вводит в строку поиска номер телефона клиента.
  1. Система выводит ФИО клиента, у которого совпадают введенные данные.

    3.1 Клиент не зарегистрирован, совпадений нет.

  1. Мастер выбирает клиента.
  1. Система выводит на экране все данные клиента: ФИО, дата рождения, телефон, адрес электронной почты.

    5.1 У клиента изменились данные.

  1. Мастер подтверждает корректность заполненных данных.
  1. Мастер добавляет клиента в заказ-наряд.

Альтернативные сценарии:

3.1 Клиент не зарегистрирован, совпадений нет.

3.1.1 Мастер реализует UC-07 “Зарегистрировать клиента”

5.1 У клиента изменились данные.

5.1.1 Мастер нажимает кнопку “Изменить”

5.1.2 Система открывает карточку клиента с полями, доступными для редактирования.

5.1.3 Мастер вносит необходимые изменения в данные клиента.

5.1.4 Мастер сохраняет внесенные данные.

5.1.5 Система выводит на экране все данные клиента: ФИО, дата рождения, телефон, адрес электронной почты.

5.1.6 Мастер подтверждает корректность данных.

5.1.7 Мастер сохраняет изменения данных.

5.1.8 Система сохраняет измененные данные.
